# Dinoki Releases

Public releases for the Dinoki app.

## What is Dinoki?

**Dinoki** is your whimsical desktop companion — a little dinosaur that roams your Mac screen and chats with you using AI. It's part productivity assistant, part emotional support dino, and fully delightful.

Learn more at [dinoki.app](https://dinoki.app)

## Download Latest Version

Visit the [Releases](https://github.com/tpae/dinoki-releases/releases) page to download the latest version of Dinoki.

## Installation

1. Download the latest `.dmg` file from the releases page
2. Open the DMG file
3. Drag Dinoki to your Applications folder
4. Launch Dinoki from Applications

## Automatic Updates

Dinoki includes automatic update functionality. Once installed, the app will notify you when new versions are available.

## System Requirements

- macOS 11.0 or later
- Apple Silicon or Intel processor
